TimeClocker 1.0

Hello, and welcome to the Time Clocker! This little application will clock you in faster than you can say "Strengthen the foundation to stimulate growth."

DO NOT close or interact with this Chrome Window!

How to use:

Enter your ExponentHR credentials into the command prompt that is open behind this window.

Once the program logs you in, you will be clocked in mon-fri 9-5 for all weeks in the current pay period. Run this program at the beginning of each period.
